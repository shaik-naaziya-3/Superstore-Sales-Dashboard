# 📊 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Sales Dashboard** developed using **Power BI Desktop** to analyze sales performance across different categories, regions, customer segments, and shipping methods. The dashboard provides business insights through KPIs, visualizations, and dynamic slicers, enabling data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales and profitability.
* Identify top-performing categories and sub-categories.
* Compare profit performance across regions.
* Understand customer segment contributions.
* Explore the impact of discounts on profitability.
* Enable users to interactively filter and explore data.

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Power Query**
* **CSV Dataset (Superstore Sales Data)**

---

## 📈 Key Performance Indicators (KPIs)

* **Total Sales:** ₹2.30M
* **Total Profit:** ₹286.40K
* **Total Quantity Sold:** 38K
* **Profit Margin:** 12.47%

---

## 📊 Dashboard Visualizations

* Sales by Category
* Profit by Region
* Sales Distribution by Segment
* Sales by Sub-Category
* Profit by Sub-Category
* Discount vs Profit Analysis (Scatter Plot)
* Quantity Sold by Category
* Ship Mode Distribution

---

## 🎛️ Interactive Features

The dashboard includes slicers for:

* Region
* Category
* Ship Mode

These filters allow users to dynamically explore the data and gain deeper insights.

---

## 📌 Business Insights

* Technology category generated the highest sales.
* Certain regions contributed more significantly to overall profit.
* Consumer segment accounted for a major share of sales.
* Some sub-categories produced high sales but relatively lower profits.
* Discount patterns influenced profitability across product segments.
